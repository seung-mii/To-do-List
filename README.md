# To-do List
SpringBoot와 React로 만든 To-do List 시스템

<br />

## 1. 제작 기간 
- 2023.03.31 ~ 2023.04.11

<br/>

## 2. 사용 기술
- Frontend 
  - CSS
  - JavaScript
  - React 
- Backend
  - SpringBoot 
  - H2 Database
  - Maven
  - Lombok
<br/>

## 3. 실행 방법
 1. SpringBoot 서버 실행 
 2. ```cd frontend``` 후  ```npm install``` 를 입력하여 node_modules 다운 
 3. ```npm start``` 를 입력하여 React 실행
 4. http://localhost:3000 접속
<br/>
